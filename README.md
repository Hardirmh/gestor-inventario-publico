# Gestor de Inventario

## 1. Descarga y Uso
1. Descarga el código (Clone o ZIP).
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta el programa principal. Si no hay base de datos, el sistema creará una vacía automáticamente.

## 2. Actualizar cambios
Si modificas el código, usa estos comandos para guardar en la nube:
   - git add .
   - git commit -m "Descripción del cambio"
   - git push

## 3. Generar ZIP limpio para IA
Para crear un ZIP sin basura ni datos sensibles (ideal para compartir con ChatGPT/Claude):
1. Ejecuta: `python preparar_proyecto.py`
2. Se creará el archivo `mi_proyecto_limpio.zip`.
